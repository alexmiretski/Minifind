# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/e75dcc9a-c9d9-4ec8-a9d1-4aed7cc5f87d

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

The app now ships with a built-in minifigure catalog and search, so it works locally without configuring an API key.
